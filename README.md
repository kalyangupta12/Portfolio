## How it was built

### Step 1 — Scaffold from image via Imagine

The initial project structure was generated by **[Imagine](https://imagine.build)** — a tool that scaffolds a full TanStack Start application from a design screenshot.

The input was an image of **Appwrite's portfolio tag page**. Imagine used this as the visual reference to produce the base layout language: editorial typographic hierarchy, structured columns, mono-heavy labels, and high contrast. It also scaffolded the TanStack Router file tree, Appwrite auth integration, and boilerplate server functions.

### Step 2 — Design system, hand-written

The Appwrite-inspired scaffold was stripped of its default theme and rebuilt with a custom CSS design system — **"Warm Editorial Minimal"** — written entirely in `portfolio.css` without Tailwind:

- Warm off-white base (`#F6F3EE`) on near-black (`#0D0C0B`)
- Typefaces: `Editorial New` (serif display) + `IBM Plex Mono` (mono body/labels)
- Single amber accent (`#8C6A42`), used sparingly
- Every CSS class is purpose-written — no component library for portfolio UI

### Step 3 — Built iteratively with Claude

All section content, component logic, animations, and feature additions were built through conversation with **[Claude](https://claude.ai) (Sonnet 4.6)** via Claude Code:

- Per-character staggered name animation on hero load (Framer Motion variants)
- Parallax scroll fade on hero via `useScroll` + `useTransform`
- Live Discord presence in footer via **Lanyard WebSocket API** — Spotify album art, VSCode activity with current file, multiple simultaneous activities
- Count-up stat animation (no library, RAF loop with cubic ease)
- `whileInView` scroll-triggered reveals on all sections
- Hover gradient overlay on project cards (`::before`, left-to-right warm fade)
- Project cards with live and source code links
- Fully responsive single-column layout on mobile

## Stack

| Layer | Tech |
|---|---|
| Framework | [TanStack Start](https://tanstack.com/start) — React 19, file-based routing |
| Bundler | Vite (rolldown-vite) |
| Language | TypeScript 5 |
| Animations | [Framer Motion](https://www.framer.com/motion/) |
| Styling | Custom CSS (hand-written design system) |
| Discord presence | [Lanyard API](https://github.com/Phineas/lanyard) — WebSocket |
| Backend scaffold | [Appwrite](https://appwrite.io) |
| Runtime | Bun |
